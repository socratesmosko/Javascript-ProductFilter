# Javascript Product Filter

This is a JavaScript-based application that allows users to search for products based on specific criteria. The application provides a user-friendly interface with a search bar and filter options. Users can filter products by name, category, price, and other attributes.

The application is designed to be lightweight and easy to use. It does not require any external libraries or dependencies, making it easy to integrate into any web application.

Technologies Used:
- JavaScript
- HTML
- CSS
